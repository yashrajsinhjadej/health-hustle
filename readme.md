## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/yashrajsinhjadej/health-hustle.git
cd health-hustle
```

2. Install dependencies
```bash
npm install
```

3. Start the application
```bash
npm start
```

### Development

To run in development mode with auto-reload:
```bash
npm run dev
```
